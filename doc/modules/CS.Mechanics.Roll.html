Functions

Roll (roll_type, lower, upper, mod, stat[, tf])
Makes a die roll.

Parameters:

  • roll_type number The type of roll, from the RollType table.
  • lower number The lower bound of the roll.
  • upper number The upper bound of the roll.
  • mod number The modifier to add on top of the roll.
  • stat string The name of the attribute rolled with.
  • tf function A function takes a number and outputs a number, that will transform the roll result. If not specified, the roll result is not transformed. (optional)

Settings

  • raid_rolls_enabled Whether or not to show roll results in the party/raid chat.
  • display_raw_rolls Whether or not to explicitly show the results of unmodified rolls.
