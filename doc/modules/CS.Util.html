iformat (format[, ...])
Formats a string with values in an order independent way. The format string may contain directives of the form %n$t where n is the index of the argument (not counting the format string itself) to insert in the directive's place, and t is a letter that indicates how to format the argument; the latter is the same as the standard string.format function. For example, the directive %2$d inserts the second argument as a decimal number.

Parameters:

  • format string The format string.
  • ... The values to format the string with. (optional)
multiline (str)
Removes indentation and initial line breaks from a string. Removes all space and tab characters following line breaks, and all line breaks at the beginning of the string. This can be used to be able to format multiline strings in source code to fit the rest of the code, without the formatting affecting the actual string value.

Parameters:

  • str string The string to process.

Returns:

    string The processed string.
compare_versions (a, b)
Compares two version strings. Compares the two version strings and returns a value which indicates which version is later. The version strings must be of the format "x.y.z" where the numbers x, y and z are the major version, minor version and patch version respectively.

Parameters:

  • a string The first version string.
  • b string The second version string.

Returns:

    number A number that is equal to 0 iff a = b, less than 0 iff a < b and greater than 0 iff a > b.
